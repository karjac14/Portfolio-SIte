Ziptrip - Road Trip Generator
Ziptrip is a road trip generator app that was presented during our Demo day in The Iron Yard. I actively contributed to this project specializing both in the functionality and the overall design, as well as in planning and interacting with the server-side (Django). Ziptrip is a single page application which makes it snappy. We utilized AngularJS, jQuery, vanilla JavaScript, SASS/CSS and HTML5 to build this wonderful app. Give it a try! Visit: Site Source
DYEL - Personal Stat Tracker
DYEL, in short for ‘Do You Even Lift’ is mini web app that can help you monitor your daily stats such as push-ups, sit-ups and or any form of exercise. We utilized AngularJS, jQuery, vanilla JavaScript, SASS/CSS and HTML5 to form this helpful app. We are in midst of updating this app, and we’ll let you know when it is ready. Site Source
Catalan Opening - MVC Simulation
The legendary chess game played by Karpov and Kasparov in 1984 can be simulated in this tiny app. This is our midterm assignment in TIY which aims to simulate an MVC framework. The model is represented by a JS file that holds the data (chess moves) and functions to execute the moves. The controller is represented by mostly jQuery and raw JavaScript. The view is of course represented by HTML and SASS/CSS. It was very fulfilling to solve this assignment as I had a lot of Aha! moments here. Give it a try! Visit: Site Source
